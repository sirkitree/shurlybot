==========
 shUrlyBot 
==========

--------------------------------------------------------------------------------------------
 shUrlyBot is a bot plugin for the bot module that allows your IRC bot to create short URLs.
--------------------------------------------------------------------------------------------

Dependencies:
 1. The bot module (http://drupal.org/project/bot)
 2. A site running shUrly (http://github.com/Lullabot/shurly.git)

The bot is currently hardcoded to use lbt.me, but will soon have options for other services.
